## 青春期后的周期（AFAB）：激素水平与时序全景

> 说明：如下数值为常见参考区间；不同实验室方法与单位可能略有差异（解读需结合检验所给参考范围与取样时点）。


## 卵巢周期与子宫内膜周期的耦合
- **卵泡期 ↔ 增生期**：FSH 轻升促卵泡群进入竞争；颗粒细胞在 FSH 作用下增强芳香化能力，使膜细胞在 LH 诱导下生成的雄激素转化为雌二醇（E2）。E2 推动内膜增厚并使宫颈黏液“雌激素化”。[^1]  
- **排卵**：当外周 E2 高水平持续达到阈值与时长要求时，负反馈短暂转为**正反馈**，GnRH/LH 出现峰值（LH surge），触发排卵。[^1]  
- **黄体期 ↔ 分泌期**：排卵后黄体分泌**孕酮（P4）为主、伴 E2**，促内膜转为分泌表型并开启“植入窗”；若未妊娠，黄体退化、P4/E2 撤退引发月经。[^1]


## 激素水平随相位的典型变化（常见参考）
- **雌二醇 E2（pg/mL）**：卵泡早期 ~20–100；**排卵前峰 ~150–750**；黄体期 ~30–450。作用：促内膜增生、诱导 LH 峰（在持续高 E2 时）。[^2]  
- **孕酮 P4（ng/mL）**：卵泡期 **<1**；**中黄体 ~5–20**（临床常以中黄体 **>3** 作为已排卵的生化证据之一）。作用：升温、促分泌期、维持早孕。[^3][^9]  
- **黄体生成素 LH（IU/L）**：卵泡期 ~1.7–15；**中期峰 ~21.9–56.6**；黄体期 ~0.6–16.3。作用：触发排卵与黄体化。[^4]  
- **卵泡刺激素 FSH（mIU/mL）**：卵泡期 ~1.4–9.9；排卵峰 ~6.2–17.2；黄体期 ~1.1–9.2。作用：招募/选择卵泡、促 E2 合成。[^5]  
- **抑制素 Inhibin**：**B 型**主导卵泡期并抑制 FSH；**A 型**主导黄体期，于中黄体达峰。[^6]

> 注：抑制素 B（卵泡期）与抑制素 A（黄体期）的时相分布，是解释“FSH 阈值/窗口”与“单一优势卵泡”现象的关键旁分泌机制之一。[^6]


## 按相位梳理“时间—事件—数值”
### 卵泡早期（经期至约 D7）
- **FSH 轻升**（常见 1.4–9.9）推动一簇小窦卵泡进入竞争；  
- **E2 低位上行**（~20–100）；**P4 低**（<1）；  
- **Inhibin B** 上升开始压低 FSH，为“选择”与“优势卵泡”创造条件。[^1][^2][^5][^6]

### 卵泡中后期（约 D8–D13）
- 形成**优势卵泡**（对降低的 FSH 仍敏感）；  
- **E2 显著上升**，FSH 因 **E2+Inhibin B** 负反馈**回落**；  
- **P4 仍低**，晚卵泡可因颗粒细胞黄体化**轻度上扬**。[^1]

### 排卵窗（约 D13–D15）
- **E2 高水平持续 ≥约 36–48 小时**达阈值 → **LH 峰**触发排卵；  
- **LH 峰起到排卵**常见 **~34–36 小时**；以“峰顶”为参照，**峰顶后 ~10–12 小时**排卵；不同研究定义带来**22–56 小时**的群体变异。[^7][^8][^1]

### 黄体期（约 D15–D28）
- **P4 为主导**：中黄体 **~5–20 ng/mL**；**E2 次峰**（数十至数百 pg/mL）；  
- **FSH/LH 受负反馈**维持低位；  
- 长度相对恒定：**平均 ~14 天**（常见 10–17 天）。未妊娠→黄体退化→P4/E2 撤退→月经。[^3][^1][^10]

### 月经与再生
- **孕酮撤退**触发：螺旋动脉痉挛、前列腺素与 MMPs 介导的组织分解 → 功能层脱落成经血；此后**迅速止血并开始再生**，进入新周期。[^1]

## “正常范围”与个体差异
- 常用参数（成人）：**周期 24–38 天**、**经期 ≤8 天**、经量通常 <80 mL；同一人最短与最长周期差 **≤7–9 天**可视为规则。[^11]  
- 青春期早期与围绝经期常见无排卵与变异；能量缺口、压力、甲状腺/泌乳素异常、PCOS 等亦可打乱节律（异常子宫出血 AUB 的常见病因谱）。[^1]


## 监测与解读要点
- **读“相位”而非单点**：相同数值在不同相位意义不同（如 P4 在卵泡期<1 生理、在中黄体<3 需警惕黄体不足/未排卵可能）。[^9]  
- **工具组合**：  
  - **尿 LH 试纸（OPK）**提示峰值临近，但**峰起到排卵间隔可变**（~34–36 h；峰顶后 ~10–12 h），宜与**中黄体 P4**和**超声**联合评估。[^7][^8][^9]  
  - **基础体温与宫颈黏液**：P4 升温与雌激素化黏液的时序特征，可作为辅助时相证据。[^1]
